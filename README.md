# Sweet Oven Bakery

## 📌 Layihə haqqında

Sweet Oven Bakery şəhər daxilində fəaliyyət göstərən müasir şirniyyat və çörək mağazası üçün hazırlanmış sadə veb sayt layihəsidir. Bu layihə vasitəsilə istifadəçilər mağaza haqqında məlumat əldə edə, məhsullarla tanış ola və sifariş imkanlarını öyrənə bilərlər.

## 🎯 Məqsəd

* Təzə və keyfiyyətli məhsulların təqdim edilməsi
* Müştərilərə rahat və anlaşılan məlumat vermək
* Onlayn sifariş imkanını göstərmək
* Gələcəkdə genişlənə bilən sistem qurmaq

## 👥 Hədəf auditoriya

* Ailələr
* Tələbələr
* Ofis işçiləri
* Şirniyyat sevənlər

## 💰 Gəlir modeli

* Günlük satış
* Xüsusi sifariş tortlar
* Onlayn sifariş və çatdırılma

## ⚙️ Funksiyalar

* Ana səhifə (Home)
* Menü səhifəsi (Məhsullar)
* Əlaqə səhifəsi (Contact)

## 🛠 Texnologiyalar

* HTML
* CSS
* Git və GitHub

## 🚀 Nəticə

Bu layihə GitHub platformasında hazırlanaraq versiya idarəetməsi tətbiq edilmişdir. Layihə istifadəçilərə sadə və rahat interfeys vasitəsilə məhsullar haqqında məlumat əldə etməyə imkan verir və gələcəkdə daha da inkişaf etdirilə bilər.
